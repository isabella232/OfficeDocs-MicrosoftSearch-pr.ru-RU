# <a name="view-connection-details-and-errors"></a>Просмотр сведений о подключении и ошибок

Чтобы получить доступ к соединитетелям и управлять ими, необходимо назначить вас администратором поиска для клиента. Обратитесь к администратору клиента для предоставления вам роли администратора поиска.

Перейдите на [вкладку Соединители](https://admin.microsoft.com/Adminportal/Home#/MicrosoftSearch/Connectors) в [Центр администрирования Microsoft 365](https://admin.microsoft.com).

Сведения о подключении и ошибки можно просматривать при нажатии кнопки подключения на вкладке [Соединители.](https://admin.microsoft.com/Adminportal/Home#/MicrosoftSearch/Connectors)  

## <a name="view-your-last-crawl-info"></a>Просмотр последней информации обхода

После успешного завершения первого начального инкрементного или полного обхода последние значения данных обхода отображаются в последнем заголовке обхода в области детализации. Если последнего обхода не было, вы не увидите никаких сведений в последнем загонах обхода. Эти сведения о последнем обходе помогут вам получить представление о том, как выполняется обход, и принять необходимые меры, где это требуется.

Для каждого подключения будут доступны следующие значения последнего обхода:

Значение | Описание
--- | ---
**Завершено в** | Дата и время завершения последнего обхода
**Тип** | Дополнительный или полный обход
**Duration** | сколько времени прошло для завершения последнего обхода
**Успешные операции** | Количество элементов, успешно взламывалось в последнем обходе
**сведения об ошибках**; | Количество элементов, которые ошиблись в последнем обходе

## <a name="monitor-errors"></a>Мониторинг ошибок

Для каждого **активного соединиттеля** на вкладке **Соединители** все существующие ошибки обхода показываются в вкладке **Ошибка.** На вкладке перечислены коды ошибок, количество каждого из них и параметры загрузки журнала ошибок. Пример см. в следующем изображении. Выберите код **ошибки,** чтобы просмотреть сведения об ошибке.

![Список соединитений с выбранным соединитетелем и области сведений, показывающая 3 ошибки для этого соединитетеля.](media/errormonitoring1.png)

Чтобы просмотреть конкретные сведения об ошибке, выберите код ошибки. Появляется экран с сведениями об ошибках и ссылкой. Последние ошибки отображаются в верхней части. Пример см. в следующей таблице.
